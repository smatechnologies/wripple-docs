# Send Email

You can use this action to send an email in your email provider.

## Input Fields

- **To** *(Required)*
- **Subject** *(Required)*
- **Message** *(Required)*
- **Send email as HTML**
- **CC**
- **BCC**
- **Reply-To Email** *(Email address that you want the respondent to reply too)*
- **Reply-To Name**

## Output Fields

This action outputs following fields:

- **Message**
- **Subject**
- **Successful**
