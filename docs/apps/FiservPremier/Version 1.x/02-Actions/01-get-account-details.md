# Get Account Details

You can use this action to get account details on Fiserv Premier.

## Input Fields

- **Account ID** *(Required)*
- **Account Type** *(Required)*

## Output Fields

This action outputs following fields:

- **Closed Date**
- **Description**
- **Id**
- **Last Statement Date**
- **Maturity Date**
- **NAICS**
- **Open Date**
- **Originating Branch**
- **Rate**
- **Responsible Branch**
- **Status**
- **Status Date**
- **Tax Exempt**
- **Tax Identity**
- **Tax Identity Type**
- **Tax Incentive Type**
- **Term**
- **Term Unit**
- **Type**
- **Withholding Option**
