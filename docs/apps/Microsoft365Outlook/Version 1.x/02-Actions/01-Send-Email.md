# Send Email

You can use this action to send an email in Outlook.

## Input Fields

- **ToRecipients**
- **CCRecipients**
- **BccRecipients**
- **ReplyToName**
- **ReplyToEmail**
- **Email Subject**
- **EMail Message**
- **Send EMail As Hmtl**


## Output Fields

This action outputs following fields:

- **Send Succesful**
- **Subject**
- **Message**
