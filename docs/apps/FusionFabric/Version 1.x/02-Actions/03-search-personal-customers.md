# Search Personal Customers

You can use this action to search for personal customers in Fusion Fabric.

## Input Fields

- **Identification Number**
- **First Name**
- **Last Name**
- **Date of Birth**
- **Phone Number**
- **Email Address** *(Required)*

## Output Fields

This action outputs following fields:

- **Items**
- **Links**
