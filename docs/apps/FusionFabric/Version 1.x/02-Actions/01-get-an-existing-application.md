# Get An Existing Application

You can use this action to get details about an existing application in Fusion Fabric.

## Input Fields

- **Application ID** *(Required)*

## Output Fields

This action outputs following fields:

- **Applicants**
- **Application Date**
- **Application ID**
- **Branch ID**
- **Category**
