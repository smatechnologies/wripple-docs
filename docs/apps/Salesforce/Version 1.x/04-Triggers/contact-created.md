# Contact Created

Use this trigger to kickstart a workflow when a new contact is created on Salesforce.

## Output fields
This trigger outputs following fields:

- **Id** (*Id of the updated account.*)
- **AccountId**
- **Email**
- **FirstName**
- **LastName**
- **MailingStreet**
- **MailingCity**
- **MailingState**
- **MailingPostalCode**
- **MailingCountry**
- **MobilePhone**
- **Phone**
