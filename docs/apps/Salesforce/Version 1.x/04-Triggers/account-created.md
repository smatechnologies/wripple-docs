# Account Created

Use this trigger to kickstart a workflow when a new account is created on Salesforce.

## Output fields
This trigger outputs following fields:

- **Id** (*Id of the newly created account.*)
- **Name**
- **BillingStreet**
- **BillingCity**
- **BillingState**
- **BillingPostalCode**
- **BillingCountry**
- **Phone**
- **Type**
