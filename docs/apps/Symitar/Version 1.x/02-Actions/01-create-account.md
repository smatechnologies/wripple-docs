# Create Account

You can use this action to create an account in Symitar.

## Input Fields

- **First Name** *(Required)*
- **Middle Name**
- **Last Name** *(Required)*
- **Email**
- **Mobile Phone**
- **Home Phone**
- **Work Phone**
- **Birth Date**
- **Occupation**
- **Street**
- **City**
- **State**
- **Country**
- **Zip Code**
- **Branch**
- **Created At Branch**
- **Open Date**
- **Close Date**

## Output fields

This action outputs following fields:

- **Account Number**
