# Override Requested

You can use this trigger to start your workflow when a security override is requested on KeyStone.


## Input Fields

N/A

## Output Fields

This trigger outputs the following fields:

- **User**
- **User Serial**
- **Override User**
- **Override User Serial**
- **Override Explanation**
- **Device Serial**
- **Request Time**
- **Last FM Date**
