# Update Event

You can use this action to update an Event in Synapsys.

## Input Fields

- **Event ID**
- - **Event Type** *(Required)*
  - Options: `Customer Contact`, `Sale`, `Service`
  - Note: This is used ONLY to find the existing note. The event type will not be updated.
- **Description**
- **Comment**
- **Status Code**
- **Due Date**

## Output Fields

N/A
