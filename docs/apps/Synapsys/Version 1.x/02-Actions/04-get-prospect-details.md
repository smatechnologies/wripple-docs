# Get Prospect Details

You can use this action to get details about a prospect in Synapsys.

## Input Fields

- **Prospect ID** *(Required)*

## Output Fields

This action outputs the following fields:

- **Prospect ID**
- **Customer Type**
- **Tax ID**
- **First Name**
- **Middle Name**
- **Last Name**
- **Com Name**
- **Employer Name**
- **Birth Date**
- **Email**
- **Home Phone**
- **Work Phone**
- **Cell Phone**
- **Street Address 1**
- **Street Address 2**
- **City**
- **State Code**
- **Postal Code**
