# Add Event

You can use this action to add an Event in Synapsys.

## Input Fields

- **Customer ID** *(Required)*
- **Event Type** *(Required)*
  - Options: `Customer Contact`, `Sale`, `Service`
- **Description**
- **Comment**
- **Status Code** *(Required)*
- **Category Code** *(Required)*
- **Due Date**

## Output Fields

This action outputs the following fields:

- **Event ID**
