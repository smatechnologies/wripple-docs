# Search Companies

You can use this action to search for companies in your HubSpot account.

## Input Fields

- **Name** *(Required)*
- **Domain**

## Output Fields

This action outputs following fields:

- **Companies**
  - **City**
  - **Create Date**
  - **Description**
  - **Domain**
  - **ID**
  - **Industry**
  - **Last Modified Date**
  - **Linked In**
  - **Name**
  - **State**
  - **Type**
  - **Zip**
