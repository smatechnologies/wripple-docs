# Create a Contact

You can use this action to create a contact in your HubSpot account.

## Input Fields

- **First Name** *(Required)*
- **Last Name**
- **Email**
- **Website**
- **Company**
- **Phone**
- **Address**
- **City**
- **State**
- **Zip Code**

## Output Fields

This action outputs following fields:

- **ID**
