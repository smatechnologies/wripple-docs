# Company Created

You can use this trigger to start your workflow when a company is created in your HubSpot account.

## Input Fields

N/A

## Output Fields

This trigger outputs following fields:

- **City**
- **Create Date**
- **Description**
- **Domain**
- **ID**
- **Industry**
- **Last Modified Date**
- **Linked In**
- **Name**
- **State**
- **Type**
- **Zip**
