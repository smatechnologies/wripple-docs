# Contact Updated

You can use this trigger to start your workflow when a contact is updated in your HubSpot account.

## Input Fields

N/A

## Output Fields

This trigger outputs following fields:

- **Address**
- **City**
- **Country**
- **Create Date**
- **Email**
- **First Name**
- **Id**
- **Last Modified Date**
- **Last Name**
- **Phone**
- **State**
- **Zip**
